Explicação Caracteres Especiais

Caracteres Especiais são usados para funções especificas, para usalos no hmtl é preciso encontrar o seu codigo, alguns exemplos são: €, para usar o simbolo de euro, §, para usar o simbolo de parágrafo e o × para usar o de multiplicação
